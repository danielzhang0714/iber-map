# BooozoO — Game Loser Detection System  
A fictional web app that tracks a user's location after they troll their teammates.  
Built using MapboxGL and the Geolocation API.

---

## Project Overview  
BooozoO simulates a “post-match penalty system” that locates users in real time.  
The animated background is designed to mimic police siren lights to increase the sense of urgency.

---

## Features  
- Real-time device location using the Geolocation API  
- MapboxGL map centered on the user with a marker  
- Police-siren-style animated gradient background  
- Retro arcade font for a comedic punishment theme  
- Clean, simple layout following basic design principles

---

## Technologies  
HTML5, CSS3, JavaScript (ES6), Mapbox GL JS, Geolocation API
